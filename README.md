# 周楠的个人外贸业务员网站（商务高级风格）

本网站基于 React，专为外贸业务员岗位求职打造。设计风格商务、简洁、高级，适配桌面和移动端。

## 快速开始

1. **安装依赖**

   ```bash
   npm install
   ```

2. **本地启动**

   ```bash
   npm start
   ```

3. **打包部署**

   ```bash
   npm run build
   ```
   打包后将 `build/` 目录的内容部署到你的服务器（如 NGINX/Node/静态托管等）。

## 网站结构

- 首页（个人简介与欢迎语）
- 工作经历
- 主要成就/项目
- 技能专长
- 教育背景
- 联系方式（电话/邮箱）
- 简历下载（pdf 可自定义替换）

## 修改内容

源码在 `src/` 下，图片与 PDF 简历放在 `public/`，如需改动直接替换即可。
